# where( type $nazevVlastnosti = NULL, type $podminka = NULL, mixed $hodnota = NULL, mixed $otevreneZleva = NULL, mixed $otevreneZprava = NULL )

Metoda nastaví filtr WHERE a vrací řetězec právě vytvořeného filtru. Pokud byl již filtr nastaven, přidá další podmínku s operátorem AND, například: podmínka1 AND podmínka2. Nové nastavení filtru způsobí, že každé další čtení/zápis vlastnosti objektu vyvolá nové načtení dat z databáze s novým filtrem Filtr je vytvořen prostým zřetězením názvu sloupce a podmínky. Použití: ->where("prijmeni", "= 'Novák'") vytvoří filtr "WHERE prijmeni = 'Novák', ->where ("vek", "> 20") vytvoří filtr "WHERE vek > 20". Metoda kontroluje existenci názvu sloupce v db tabulce, pro neexistující sloupec vyhodí výjimku.
